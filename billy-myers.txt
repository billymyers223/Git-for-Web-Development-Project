    1. What is Semantic HTML? 
	Semantic HTML can be seen by the user and is interpreted by the browser to display on the page
    2. What is HTML used for?
	HTML is the structure of websites, it tells the browser what to display and allows developers to format and create a site.
    3. What is an attribute and where do we put it? 
	Attributes are put within tags to describe what to do (alt, href, src, height, & width). 
    4. What is the h1 tag used for? How many times should I use it on a page?
	The h1 tag is used as a main heading on a page and is typically used once.
    5. Name two tags that have required attributes
	<img> <a>
    6. What do we put in the head of our HTML document? 
	Metadata
    7. What is an id? 
	an ID is an attribute used to specify an HTML element
    8. What elements can I add an id to? 
	You can add styling to the HTML element with that ID.
    9. How many times can I use the same id on a page? 
	Once
    10. What is a class? 
	A class is an HTML element
    11. What elements can I add a class to? 
	Any CSS elements
    12. How many times can I use the same class on a page? 
	As much as you'd like
    13. How do I get my link to open in a new tab?
	target= "_blank"
    14. What is the alt attribute in the image tag used for? 
	It is used for screen readers to increase site asscesibility
    15. How do I reference an id?
	You use a #
    16. What is the difference between a section and a div
	A div is a block level element which is used to divide a site and a section defines a section of content on a site.
    17. What is CSS used for? 
	css is used for styling an HTML document
    18. How to we select an element? Example - every h2 on the page
	h2{Styling goes here}
    19. What is the difference between a class and an id? - Give me an example of when I might use each one
	A class is able to be used more than once in HTML which allows you to style a group of tags the same. An ID can only be used once which can help make one element stand out more than others on the page.
    20. How do we select classes in CSS?
	.ClassName{}
    21. How do we select a p element with a single class of “human””?
	You would do p .human{}
    22. What is a parent child selector? When would this be useful? 
	A parent child selector passes all the parent CSS styling down to the child, this is useful when you want to go with one unique stylign rule for that section of your code
    23. How do you select all links within a div with the class of sidebar?

    24. What is a pseudo selector?
	A pseudo selector is a keyword added to a selector that specifies a special state
    25. What do we use the change the spacing between lines?
